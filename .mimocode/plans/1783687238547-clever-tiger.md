# Plano: Sistema de Internacionalização (i18n) para 3 idiomas

## Visão Geral
Adicionar um sistema de tradução com 3 idiomas (EN, PT, IT) ao portfólio, com detecção automática do idioma do sistema e um seletor de idioma na navbar.

## Abordagem
Criar um composable Vue leve (`useI18n`) com arquivos de tradução separados — sem dependências externas (não usar `vue-i18n`, é excesso para um portfólio com 3 idiomas).

---

## Arquivos a Criar

### 1. `src/i18n/en.js` — Traduções em inglês (texto atual)
### 2. `src/i18n/pt.js` — Traduções em português
### 3. `src/i18n/it.js` — Traduções em italiano
### 4. `src/i18n/index.js` — Agrega todas as traduções
### 5. `src/composables/useI18n.js` — Composable reativo
- Detecta idioma via `navigator.language` (prefixo `pt` → PT, `it` → IT, senão → EN)
- Persiste escolha no `localStorage`
- Expõe `t(key)` reativo + `locale` + `setLocale()`
### 6. `src/components/layout/LanguageSwitcher.vue` — Botão de troca
- Botão compacto ao lado do "Download Resume" na navbar
- Mostra bandeiras ou códigos de idioma (EN / PT / IT)
- Design glass consistente com os outros botões
